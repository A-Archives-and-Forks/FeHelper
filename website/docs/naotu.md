# 思维导图工具

## 工具简介

思维导图工具是一个专业的思维导图编辑工具，基于KityMinder实现，支持创建、编辑、保存和导出思维导图。该工具提供了直观的图形化界面，帮助用户更好地组织和展示思维结构，是学习和工作的得力助手。

## 主要功能

### 1. 思维导图编辑
- 节点创建与编辑
- 节点关系管理
- 节点样式设置
- 节点图标选择
- 节点备注添加

### 2. 文件管理
- 本地保存
- 文件导出
- 历史记录
- 文件列表
- 自动备份

### 3. 编辑功能
- 拖拽操作
- 快捷键支持
- 撤销重做
- 复制粘贴
- 节点折叠

### 4. 导出功能
- JSON格式导出
- 图片导出
- 在线分享
- 批量导出
- 格式转换

## 常见问题

1. **Q: 如何保存思维导图？**
   A: 点击保存按钮或使用快捷键即可保存到本地。

2. **Q: 支持哪些导出格式？**
   A: 支持JSON格式导出，可以导入到在线版本继续编辑。

3. **Q: 如何分享思维导图？**
   A: 可以导出JSON文件，或使用在线版本分享链接。

4. **Q: 数据如何备份？**
   A: 建议定期导出JSON文件进行备份。
