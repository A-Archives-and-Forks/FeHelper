# 进制转换工具

## 简介
进制转换工具是FeHelper提供的一个简单而强大的数字进制转换工具，支持2-36进制之间的相互转换。该工具提供了直观的界面和实时转换功能，帮助用户快速完成不同进制之间的转换工作。

## 常见问题
1. **Q: 为什么某些数字转换后显示异常？**
   A: 可能是输入格式不正确或数值超出范围，建议检查输入值是否符合当前进制的格式要求。

2. **Q: 如何处理大数转换？**
   A: 工具支持最大32位的数字输入，超过此范围可能会导致精度损失。

3. **Q: 转换结果不准确怎么办？**
   A: 检查输入格式是否正确，确保数值在有效范围内，必要时可以尝试其他进制。
